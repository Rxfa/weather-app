## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

The app communicates with the [WeatherStack api](https://weatherstack.com/) to retrieve information about the weather conditions in a given location and then displays it to the user.

The user chooses a location by typing it's name in a search bar.

The info displayed is:

* City and Country name
* Weather description and icon
* Temperatures (current, min and max)

## Getting Started

### Prerequisites

* npm

  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Get a free API Key at [https://weatherstack.com](https://weatherstack.com)
2. Clone the repo

   ```sh
   git clone https://github.com/rxfa/weather-app.git
   ```

3. Install NPM packages

   ```sh
   npm install
   ```

4. Enter your API with [django-rest-framework-api-key](https://florimondmanca.github.io/djangorestframework-api-key/)
